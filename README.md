# Task App - PWA Task Manager

A beautiful, mobile-first task management app built with React. Perfect for organizing your day!

## Features

✨ **Core Features:**
- ✅ Complete tasks with one swipe
- 📅 Schedule tasks for any date
- 🔄 Recurring tasks (daily, weekly, monthly)
- 🏷️ Tag your tasks (home, school, misc)
- 🔍 Filter and sort by tags
- 📑 Subtasks for breaking down big tasks
- 📊 Progress bar tracking daily completion
- ⏰ Set reminders for important tasks
- 💾 Offline-first with local storage

## Swipe Gestures

- **Swipe Right** → Mark task as complete
- **Swipe Left** → Reschedule task
- **Long Press** → Highlight task for actions

## How to Deploy to GitHub Pages

1. **Fork or Clone this repository:**
   ```
   git clone https://github.com/YOUR-USERNAME/task-app.git
   cd task-app
   ```

2. **Push to GitHub:**
   ```
   git add .
   git commit -m "Initial commit: Add task app"
   git push origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click **Settings** → **Pages**
   - Under "Source", select **Main branch**
   - Click **Save**
   - Wait 1-2 minutes for deployment

4. **Access your app:**
   - Your app will be live at: `https://YOUR-USERNAME.github.io/task-app`
   - Bookmark this URL or add it to your home screen!

## Technical Details

- **Built with:** React 18 + Tailwind CSS
- **Data Storage:** Browser localStorage (all your tasks saved locally)
- **Offline Support:** Works completely offline
- **Browser Support:** Modern browsers (Chrome, Safari, Firefox, Edge)
